# Learn doctrine with tdd
通过 TDD(测试驱动开发) 学习 Doctrine

# 命令
生成迁移类
```shell script
vendor/bin/doctrine-migrations generate
```

批量执行迁移
```shell script
vendor/bin/doctrine-migrations migrate --no-interaction
```
